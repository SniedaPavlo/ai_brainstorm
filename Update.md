Модератор между раундами — после каждого раунда (кроме последнего) модератор анализирует дискуссию, выделяет конфликты и слепые зоны, и формулирует конкретные вопросы для следующего раунда. Агенты обязаны отвечать на эти вопросы.
Поток теперь такой:
Раунд 1 → агенты отвечают → модератор выделяет конфликты → Раунд 2 → агенты отвечают с фокусом модератора → модератор → ... → Финальный раунд → action-plan с приоритетами P0/P1/P2 и сроками.
